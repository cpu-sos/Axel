ESTE ES EL EJEMPLO PARA MODIFICAR EL ARCHIVO

Para publicar un sitio web en Github, primero se necesita crear una cuenta en el sitio, después de esto se crear un repositorio, en este se debe crear un archivo en la raíz llamado "index.html". Al finalizar esto, se debe descargar un editor de código, como Visual Studio Code, y se debe crear una carpeta para recopilar los archivos, y después se debe abrir el archivo index.html y escribir la estructura básica de código. Finalmente, se añaden los elementos deseados al código y se arrastra el archivo al repositorio.
